# 时间轴重构回归检查清单

## 时间模型与显示
- 2x：同一片段在时间轴上长度缩短为 1/2，刻度与播放头位置一致。
- 0.5x：同一片段在时间轴上长度变长为 2 倍，拖拽起止点仍对齐刻度。
- 裁剪 + 倍速：显示时长 = (裁剪时长 / 倍速)。

## 交互行为
- 左/右裁剪拖拽：调整量与时间轴秒数一致（倍速下不会过快/过慢）。
- 片段分割：在播放头位置分割后，两段边界与播放头一致。
- 片段合并：相邻片段可合并，非相邻提示错误。
- 磁吸：片段边界/播放头的吸附点按可见时长计算。

## 播放与导出
- 播放：倍速下时间轴时间推进符合可见时长，音视频同步。
- 导出进度：进度条总时长与时间轴可见时长一致。
